## Tool Integrations

Integrate Tonik workflows with your favorite design and development tools. Use these examples to connect seamlessly.

<Tabs>
  <Tab title="Figma" icon="feather">
    Export designs directly to Tonik for prototyping.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const figmaToken = 'YOUR_FIGMA_TOKEN';
      const response = await fetch('https://api.figma.com/v1/files/{file_key}', {
        headers: { 'X-Figma-Token': figmaToken }
      });
      const designs = await response.json();
      // Send to Tonik API
      ```
      ```python
      import requests
      figma_token = 'YOUR_FIGMA_TOKEN'
      response = requests.get(
        'https://api.figma.com/v1/files/{file_key}',
        headers={'X-Figma-Token': figma_token}
      )
      designs = response.json()
      # Integrate with Tonik
      ```
    </CodeGroup>
  </Tab>
  <Tab title="Webflow" icon="globe">
    Deploy no-code sites with Tonik branding kits.

    ```javascript
    // Webflow CMS integration example
    const siteId = 'YOUR_SITE_ID';
    fetch(`https://api.webflow.com/v2/sites/${siteId}/cms/collections`, {
      headers: {
        'Authorization': `Bearer YOUR_WF_TOKEN`,
        'accept-version': '1.0'
      }
    }).then(response => response.json());
    ```
  </Tab>
  <Tab title="Zapier" icon="zap">
    Automate workflows between Tonik and your stack.

    <Steps>
      <Step title="Connect Zapier" icon="plug">
        Create a Zap with Tonik as the trigger app.
      </Step>
      <Step title="Add Action" icon="play">
        Select tools like Slack or Airtable for actions.
      </Step>
      <Step title="Test & Deploy" icon="check-circle">
        Run a test and activate your Zap.
      </Step>
    </Steps>
  </Tab>
</Tabs>
